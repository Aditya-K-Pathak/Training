Projects

Object Detection System for Visually impaired Persons

The project is using a deep learning algorithm YOLO v5 which is about 2.5 times faster than other CNN algorithms
As output, it tells the user the name of object present in front of the user/camera
It can be used as motion sensor camera of surveillance camera

Monument Identification and detailing device

The project is using a deep learning algorithm YOLO v5 which is about 2.5 times faster than other CNN algorithms
As output, it tells the user the ndetails of the Monument such as the place where it is situated, current temparature of the place, distance of the monument from user's current location etc.
Other than these, it can be used as an assistant by commomn people or by local guides to promote tourism

Health & Habit Tracker

The project receives some useful insights such as activity minutes, heart rate from Google Fit API, location info from IP address and and store these data in a database.
These data are later used to extract information like Blood-Pressure, Sugar Level and Pulse rate of the user in real time
